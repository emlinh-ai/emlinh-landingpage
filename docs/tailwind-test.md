# Tailwind CSS Test

Đây là file tài liệu để kiểm tra Tailwind CSS đã được tích hợp thành công.

## Cách kiểm tra:

1. Chạy development server: `npm run dev`
2. Mở trình duyệt và xem ứng dụng
3. Kiểm tra các class Tailwind trong file App.tsx

## Class Tailwind để thử:

- `bg-blue-500` - Nền màu xanh dương
- `text-white` - Chữ màu trắng  
- `p-4` - Padding 1rem
- `rounded-lg` - Bo góc
- `shadow-lg` - Bóng đổ

## Ví dụ sử dụng:

```jsx
<div className="bg-blue-500 text-white p-4 rounded-lg shadow-lg">
  <h1 className="text-2xl font-bold">Tailwind CSS hoạt động!</h1>
</div>
```
